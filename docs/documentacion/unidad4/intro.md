# Órbita 4: Pulir para que brille
## Introducción

Has terminado de maquetar tu proyecto. Tienes una estructura HTML sólida, tus estilos CSS están organizados, el responsive funciona. Ahora llega el momento de la verdad: añadir las imágenes y vídeos que van a dar vida visual a tu interfaz.

Pero aquí viene el problema que muchos desarrolladores junior no anticipan: si simplemente arrastras las imágenes que te pasó el diseñador y las insertas con un `<img src="foto.jpg">`, tu web puede convertirse en un desastre de performance. Una sola imagen hero sin optimizar puede pesar más que todo tu código JavaScript. Un vídeo de fondo mal preparado puede hacer que los usuarios móviles agoten sus datos en segundos.

Esta órbita trata de preparar y optimizar recursos multimedia de forma profesional. Vamos a aprender a elegir el formato correcto para cada caso, optimizar el peso sin sacrificar calidad, servir diferentes versiones según el dispositivo, y documentar todo el proceso.
